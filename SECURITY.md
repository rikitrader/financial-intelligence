## ACTIVATION PROCESS

### Step 1: Clone the Repository

```bash
git clone https://github.com/rikitrader/financial-intelligence.git
cd financial-intelligence
```

### Step 2: Run the Activation Script

```bash
./scripts/activate.sh
```

### Step 3: Enter License Key

When prompted, enter the license key provided by the copyright holder.

By entering the license key, you agree to:

## ACTIVATION FILE

Upon successful activation, a `.activated` file is created containing:
- Activation status
- Activation date
- Username

**Do NOT delete this file** - you will need to reactivate if removed.

**Do NOT share this file** - it is local to your installation.

---

## DEACTIVATION

To deactivate the license:

```bash
rm .activated
```

You will need to reactivate to use the software again.
